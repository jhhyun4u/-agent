## 3. 코드 품질 개선

### 3.1 버그 수정 (5개 HIGH 이슈)

1. **formatCurrency 로직 버그** ✅
   - 문제: 1억원 미만은 항상 "-" 표시
   - 수정: 만원 단위 포맷 추가 + 억원 계산 개선

2. **클라이언트 정렬 최적화** ✅
   - 문제: 매 렌더마다 O(n log n) 정렬
   - 수정: `useMemo` 추가로 items 변경 시만 재계산

3. **모달 접근성** ✅
   - 문제: role="dialog" 및 aria-* 속성 누락
   - 수정: WCAG 2.1 규준 준수 속성 추가

4. **테이블 접근성** ✅
   - 문제: scope="col" 및 caption 없음
   - 수정: 스크린 리더 접근성 개선

5. **페이지네이션 에러 처리** ✅
   - 문제: 에러 상태에서도 페이지 컨트롤 표시
   - 수정: `!error` 조건 추가

### 3.2 타입 안전성 강화

- `WinResultBadge` 타입 개선 (switch-case exhaustive check)
- `BidAnnouncement` 타입 확장 (suitability_score 필드)
- 불필요한 import 제거 (`useRouter`)
