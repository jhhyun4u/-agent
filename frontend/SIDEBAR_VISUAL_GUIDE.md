## 🎬 인터랙션 플로우

### 데스크톱 - 드래그 리사이즈

```
마우스 다운 (우측 경계)
    ↓
  [드래그 오버레이 표시]
    ↓
마우스 이동 (clientX 추적)
    ↓
  [너비 실시간 업데이트]
  [180px ~ 360px 범위 유지]
    ↓
마우스 업
    ↓
  [너비를 localStorage에 저장]
  [드래그 오버레이 제거]
```

### 그룹 토글

```
지식베이스 토글 버튼 클릭
    ↓
  kbOpen 상태 변경
    ↓
  localStorage에 저장
    ↓
  하위 메뉴 슬라이드 애니메이션
    ↓
  펼침/접힘 표시 변경 (▾/▸)
```

### 경로 기반 자동 펼침

```
pathname 변경 감지
    ↓
  /kb/* 로 시작하는가?
    ├─ YES → kbOpen = true (자동 펼침)
    └─ NO  → 상태 유지

  /admin/* 로 시작하는가?
    ├─ YES → adminOpen = true (자동 펼침)
    └─ NO  → 상태 유지
```

## 🔄 최근 작업 예시

```
최근 작업
├─────────────────────────────────────┐
│ 🟠 프로젝트 A (초기화 중)            │
│    D-2 · Phase 1/5                  │
├─────────────────────────────────────┤
│ 🟢 프로젝트 B (진행 중)              │
│    D-10 · Phase 3/5                 │
├─────────────────────────────────────┤
│ 🟢 프로젝트 C (진행 중)              │
│    D+5 · Phase 2/5                  │
└─────────────────────────────────────┘

범례:
🟠 = initialized (준비 중)
🟢 = processing/running (진행 중)

D-2: 마감까지 2일 남음 (빨강)
D-10: 마감까지 10일 남음 (노랑)
D+5: 마감 초과 5일 (회색)
```
